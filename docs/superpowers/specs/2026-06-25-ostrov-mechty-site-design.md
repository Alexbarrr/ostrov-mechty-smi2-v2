# Остров Мечты × СМИ2 — сайт-презентация

**Дата:** 2026-06-25
**Источник:** `Остров_Мечты_СМИ2.pptx` (11 слайдов) + находки дизайн-ревью.

## Цель
Превратить pptx в веб-презентацию на GitHub Pages с анимацией и интерактивом,
чтобы дальше быстро итерировать (правки видны в один коммит).

## Формат и стек (залочено)
- **Гибрид:** 11 полноэкранных секций (100vh), scroll-snap, навигация стрелками /
  колесом / свайпом / точками. Богатые анимации входа на каждом слайде.
- **Стек:** Vanilla HTML + CSS + JS, **без сборки**. GSAP (CDN) + ScrollTrigger
  для анимаций. Деплой — статика прямо на GitHub Pages.
- **Шрифты:** пара по ролям — **Unbounded** (дисплейные заголовки) + **Onest**
  (текст, описания, сноски). Google Fonts. Чинит находку H1 (Arial+Calibri).

## Дизайн-система
- **Фон:** тёмное индиго. `--bg:#0E0F22`, `--bg-2:#12132B`, `--surface:#1C1E3D`,
  `--surface-2:#23264A`, бордеры `#2E315C`.
- **Текст:** `--ink:#FFFFFF`, `--muted:#B9BEDC` (крупнее/контрастнее, чем в pptx —
  чинит M1/H3), приглушённый `#8A8FB8` только для мелких меток.
- **Акцент:** ОДИН бренд-оранжевый `--accent:#FF5A2C`. Лайм/зелёный из заголовков
  убраны (находка H2). Зелёный `#40AE49` допустим точечно для «положительной»
  дельты, но не в заголовках.
- **Типошкала:** дисплей 64–96px, h2 40–56px, body 20–22px, сноски 16–18px
  (всё крупнее pptx). Перенос заголовков `text-wrap: balance`.

## Сквозные элементы (на каждом слайде)
- **Лого-лок СМИ2 × Formats Media** в правом верхнем углу (комментарий заказчика).
- Прогресс-бар сверху, точки-навигация справа, футер «Остров Мечты × СМИ2» + №.
- Тёмный градиент-скрим под текстом на фото-слайдах (чинит H3).

## Правки контента
- Слово **«посев/посевы» убрано везде** → «размещения в медиа / медиа-охват»
  (слайды 5, 9, 11) (комментарий заказчика).
- Описания и сноски — крупнее и контрастнее (комментарий + M1/H3).

## 11 секций
1. **Титул** — замок (Ken Burns зум), заголовок собирается по буквам, скрим.
2. **Группа компаний СМИ2** — текст слева, фото купола справа, 3 продукта.
3. **«Не парк для детей»** — стереотип vs реальность, фото купола со скримом.
4. **Что мы решаем** — 4 задачи (сетка 2×2, контраст усилен vs M2).
5. **Продукты ГК СМИ2** — 3 формата, единый акцент, ровный ритм карточек (M3).
6. **Внутри ЦА** — 4 типа внимания, network.jpg как приглушённая текстура.
7. **Кейс 01 · ТГБ** — 3 пункта, фото купола.
8. **Кейс 02 · Рич** — счётчики ×3,75 и −66% (count-up при появлении). Якорь дека.
9. **Кейс 03 · Инфоволна** — 3 пункта, фото замка.
10. **Методология** — ТГБ + Рич + Инфоволна → «Продажа эмоции» (влёт по очереди).
11. **Выводы** — 4 буллета, фото со скримом.

## Анимации (база + 3 фирменных момента)
- База: fade/slide входы (ScrollTrigger), parallax фото, hover-объём карточек,
  плавные переходы, прогресс/точки.
- Вау-1: титул — Ken Burns + сборка заголовка.
- Вау-2: слайд 8 — count-up цифр.
- Вау-3: слайд 10 — карточки влетают и складываются в оранжевый блок с пульсом.
- `prefers-reduced-motion` уважается (анимации выключаются).
