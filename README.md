# Остров Мечты × СМИ2 — сайт-презентация (v2)

Веб-версия питч-дека «Остров Мечты × СМИ2»: работа на все группы аудитории по вниманию.

11 полноэкранных слайдов, навигация стрелками / колесом / свайпом / точками,
анимации входа, count-up метрик, parallax фото.

## Стек
Чистый HTML + CSS + JS, GSAP (CDN). Без сборки — статика для GitHub Pages.

## Локально
```bash
python3 -m http.server 8765
# открыть http://localhost:8765/
```

## Структура
- `index.html` — разметка слайдов
- `styles.css` — дизайн-система и адаптив
- `main.js` — навигация, reveal-анимации, count-up, parallax
- `assets/img/` — фото парка
- `docs/superpowers/specs/` — дизайн-спек

Шрифты: Unbounded (заголовки) + Onest (текст). Акцент — оранжевый `#FF5A2C`.
